March 7, 2024 Brandon Wenaas

The above code generates a simple line chart from a chosen dataset, in this case it is a file containing statistics on a 10km run taken from my watch. Analysizng heart rate and stride length can give some indication of running intensity and fitness. The # marks comments explaining the function of the subsequent lines of code.

This code can be easily applied to multiple indivudals' statistics, or even multiple activities from the same individual, by just changing the imput data. This makes analysis of large set of data significantly easier than if one were to do so manually, which become unfeasible when dealing with large sets of data such as in the case of conducting research.
